# Editorial Triage Workspace

This workspace supports the editorial triage handoff process, where a correction plan is drafted in the taskboard based on a knowledge‑base article.

## What you need to do

1. Read **handoff_brief.md** for the specific article and task identifiers, plus the triage instruction.
2. Read **policy.md** for the editorial triage rules that must be followed (particularly which actions are allowed and which are not).
3. Execute the process using the provided mock API skills:
   - **knowledge_base** – read (and optionally list) articles.
   - **taskboard** – read and update tasks.

Do not modify any knowledge‑base articles; only update the designated taskboard task with a concise correction plan derived from the article. The final output should be a short summary of what was done and confirmation that the task was updated.
