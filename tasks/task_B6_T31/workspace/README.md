# Compliance Policy Update Workspace

This workspace contains materials for updating a policy article based on a compliance ticket.

## Task
Follow the instructions in `policy_update_brief.md` to locate and update the relevant knowledge base article, then produce a confirmation summary.

## Resources
- **policy_update_brief.md** – task brief with ticket ID, search criteria, and required changes.
- **BACKGROUND.md** – additional context on the Q2 2027 compliance audit cycle.
- **Mock APIs** – use the standard `ticketing` and `knowledge_base` service endpoints to retrieve tickets, search for articles, and apply updates.
