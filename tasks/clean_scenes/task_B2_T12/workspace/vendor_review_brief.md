# Vendor Review Brief

This document lists the assessment questions you must answer when reviewing the Acme Vendor Solutions intake page against our internal vendor intake security policy.

## Review Questions

1. **Data Residency**: Does the vendor's stated data storage and processing comply with our internal data residency requirements as defined in the security policy?
2. **Security SLA**: Does the vendor provide a signed security SLA that meets the terms and conditions specified in our policy?
3. **Gap Analysis**: Are there any significant gaps, missing commitments, or other security-related concerns based on the vendor's intake page that should be flagged?
