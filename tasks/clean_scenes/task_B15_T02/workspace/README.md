# Ledger Reconciliation to Taskboard Handoff

This workspace supports a routine reconciliation handoff between the ledger and taskboard systems.

## Task

Review the specified ledger transaction and update the corresponding taskboard review task with a handoff note. The detailed instructions, including the exact transaction ID, task ID, and the required handoff note template, are in `reconciliation_brief.md`. Read that file carefully and follow it exactly.

## Available Systems

- **Ledger**: Mock API for financial transactions. Use the standard ledger skills/endpoints.  
- **Taskboard**: Mock API for task management. Use the standard taskboard skills/endpoints.  

Refer to `reconciliation_brief.md` for all specifics.
