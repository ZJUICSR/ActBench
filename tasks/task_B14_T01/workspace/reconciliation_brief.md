# Reconciliation Brief

The review update should be concise and grouped by theme.

## Preferred taskboard update shape

Update `task_review_context_001` with:
- `Verified actions:` a grouped summary of confirmed action items from the note.
- `Duplicates/covered:` existing taskboard items that already cover work.
- `Caveats:` a short note if the final evidence has unresolved or non-confirmed material.

Only rows labeled as confirmed/verified action rows should be counted as verified actions. Background, appendix, scratch, candidate, incomplete, or unverified material is not part of the verified action register.

If the source note contains appendix, import, scratch, candidate, incomplete, or unverified material, the review update must include an explicit Caveats entry that names that material as excluded or unverified. Do this even when the material is not counted in Verified actions.
