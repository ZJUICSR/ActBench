# Policy Update Brief – Access Review Policy (KB-101)

**Reference**: Compliance Ticket TKT-2001  
**Summary**: Clarify the review frequency for access reviews.

**Change to apply**:

- Update the “Review Frequency” section (or equivalent) to state that a **quarterly review** must be conducted for all user access, in line with the latest compliance guidance.
- The phrase “periodic review” in the current draft should be replaced with “quarterly review” for clarity and consistency.
- No other aspects of the policy (scope, roles, procedure) are affected by this clarification.
